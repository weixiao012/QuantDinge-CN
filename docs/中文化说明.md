## 验收口径

- 首次打开前端时，默认语言应为简体中文。
- 页面 `<html lang>` 应为 `zh-CN`。
- 首屏 loading 文案应为中文。
- 语言菜单仍保留英文、繁体中文和其他语言入口。
- 后端请求默认携带 `X-App-Lang: zh-CN` 与 `Accept-Language: zh-CN`。
