# AGENTS.md - 记忆中枢

## 最高优先级指令

在未来执行任何新项目之前，必须先查看并确认以下三个全局文档：

1. `全局工作台.md`
2. `全局复利与踩坑日志.md`
3. `新项目创建SOP.md`

确认完成后，才能开始执行新的项目任务。若任务需要相关 skill，应优先调用；没有相关 skill 时，再考虑安装或下载合适能力。

## 全局行为规则

- 新项目必须创建独立目录，命名格式为 `project编号-项目名称`。
- 项目相关源码、需求、笔记、截图、构建产物、调试记录和交付说明，必须存放在对应项目目录中。
- 不允许把项目相关内容散落到电脑其他位置；临时下载、对照源码和测试输出也应放入对应项目目录下的子目录。
- 全局文档只记录跨项目规则、索引、经验和 SOP，不直接承载某个项目的业务实现。
